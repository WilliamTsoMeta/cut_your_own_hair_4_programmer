# cut_your_own_hair_4_programmer

工具：

1. 理发器（推子clippers）
2. 剪刀 （scissrs）
3. 手持镜子
4. 大镜子

步骤：

1. 用4mm卡尺从前向后推两侧不超过耳朵
2. 8mm推整个后脑勺
3. 6mm从下到上推后脑勺的2/3和两侧
4. 4mm从下到上推后脑勺1/3
5. 2mm推后勃颈处修出轮廓
6. 剪子 用五只手指插入头发剪刀你认为合适的长度直到所有头发长度一致
